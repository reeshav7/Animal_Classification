# üêæ Animal Image Classification (ANN vs CNN)

This project implements an **Animal Image Classification system** that compares **Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)** for image recognition. The system includes a **GUI-based application** that allows users to upload images and obtain real-time predictions with confidence scores.

---

##  Project Overview

Image classification is a core problem in computer vision. In this project, two different approaches are explored:

- **ANN (Artificial Neural Network)** with PCA-based feature reduction  
- **CNN (Convolutional Neural Network)** using transfer learning (MobileNetV2)

The objective is to understand the limitations of traditional ANN models for image data and demonstrate why CNNs perform better for vision-based tasks.

---

##  Key Features

- Multi-class animal image classification (10 classes)
- ANN vs CNN performance comparison
- PCA and StandardScaler preprocessing for ANN
- Transfer learning using MobileNetV2 for CNN
- Interactive GUI for real-time predictions
- Confidence score visualization
- Clean and modular project structure

##  Dataset

The dataset contains images from the following **10 animal classes**:

- Butterfly  
- Cat  
- Chicken  
- Cow  
- Dog  
- Elephant  
- Horse  
- Sheep  
- Spider  
- Squirrel  

The dataset is split into approximately **74% training** and **26% testing** data.
